This project is a Python-based Password Generator and Strength Checker.
It allows users to generate strong, random passwords by providing a desired length.
Users can also input an existing password to evaluate its strength.
The strength is classified as Weak, Medium, or Strong based on security rules.
The program checks for the presence of uppercase letters, lowercase letters, numbers, and special characters.
It ensures the password meets minimum length requirements.
